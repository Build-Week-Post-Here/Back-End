# Back-End

#### User signup:

> POST https://reddit-ranker.herokuapp.com/api/auth/register

post body example:
```js
{
    "email": "shaq@me.com",
    "password": "pass123"
}
```

#### User login:

> POST https://reddit-ranker.herokuapp.com/api/auth/login

post body example:
```js
{
    "email": "shaq@me.com",
    "password": "pass123"
}
```
